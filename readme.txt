Welcome to the game. Vegetarian Snake is a classic approach in
creating a snake-type game, with basic mechanics on collision with
different objects with particular outcomes.

Features:

 - Canvas-type field 400x400 with border. Border returns GAMEOVER on
 collision (+);

 - Vegetable object, randomly appeared on the field, which makes snake
 longer after collision (+);

 - Moving snake, wich can be controlled with up,down,left,right buttons (+);

 - Different difficulty level, related to game speed (+);

 - Dynamic score (+);
